# **App Name**: Graphistory

## Core Features:

- Grid View: Display images in a responsive grid layout.
- Sphere View: Arrange images in an interactive 3D sphere where photos always face the viewer and support pan, tilt, and zoom with the mouse.
- Name View: Rearrange images to form a specified word (e.g., 'GEMINI').
- Explore View: Display a selected image zoomed in, with navigation buttons for 'Prev' and 'Next'.
- Image Description Generator: Generate a brief description for each image, displayed in Explore View. Use reasoning tool to incorporate user preferences.
- Mode Transitions: Implement smooth transitions between grid, sphere, name, and explore modes.
- Image Interaction: Allow image selection and zooming, with smooth transitions to explore mode upon clicking.

## Style Guidelines:

- Primary color: Dark, muted purple (#5A4AE3) for a modern and sophisticated feel, evocative of technology and exploration.
- Background color: Dark gray (#282A3A) to create a visually immersive experience.
- Accent color: Electric blue (#79E2F2) to highlight interactive elements and call to actions, drawing the eye and providing visual clarity against the dark background.
- Headline font: 'Space Grotesk' sans-serif, to provide a tech-inspired contemporary feel
- Body font: 'Inter', sans-serif, for readability and clean aesthetic
- Use minimalist, line-based icons for navigation and interactive elements.
- Fixed bottom navigation bar for easy mode switching.
- Subtle transitions and animations when switching between modes and interacting with images.
